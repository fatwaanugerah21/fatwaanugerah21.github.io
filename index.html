Tutorial Kumpul Tugas Grup Asistensi 3 PP 2022

  1. Download git pada link https://git-scm.com/downloads

    Klik OS sesuai yang digunakan

  2. Pilih 32 bit atau 64 bit (standalone installer) Sesuaikan dengan windows nya

  3. Install git yang telah didownload.. (next” aja)

  4. Setelah git terinstall, buat akun github pada https://github.com (jika tidak punya)

  5. Setelah memiliki akun github, buka halaman https://github.com/fatwaanugerah21/Praktikum_PP_Lab_PP_2022 lalu tekan tombol fork pada halaman tersebut

    lalu tekan "Create fork"

    Pada kotak merah akan menampilkan “username_github/Praktikum_PP_Lab_PP_2022 (username_github adalah username github masing”)

    Lalu buka “command prompt” jika OS yang digunakan windows, dan "terminal" untuk Mac dan Linux
    Tampilannya kurang lebih seperti ini (windows, linux dan mac).

    Masih pada halaman

    copy link pada halaman tersebut

  6. Lalu ketikkan “git clone https://github.com/NPEDigiHouse/Praktikum_PP_Lab_PP_2022” Link adalah link yang telah dicopy pada step sebelumnya

    Output nya jika berhasil akan seperti ini

  7. Setelah itu ketik “cd Praktikum_PP_Lab_PP_2022” lalu enter

  8. Setelah itu ketik
    “git config --global user.name "fatwaanugerah21 enter
    “git config --global user.email “fatwaanugerah21@gmail.com” enter

  9. buat folder dengan nama "nim" masing - masing di dalam folder Praktikum_PP_Lab_PP_2022 yang sudah di clone pada step no 6

  10. masukkan tugas (kode .py) ke dalam folder nim yang telah dibuat

  11. setelah itu kembali ke "command prompt" atau "terminal", lalu ketik git status

    akan terlihat tulisan folder nim berwarna merah, yang berarti ada file yang belum dikumpul

  12. setelah itu ketik "git add ."

    lalu ketik "git status" lagi

    sekarang akan terlihat tulisan file kode .py yang telah kita masukkan ke dalam folder nim berwarna hijau, yang berarti file siap dikumpulkan

  13. setelah itu buat branch dengan nama nim(sesuai nim masing - masing) dengan mengetikkan "git branch H071191040"

    lalu ketik "git checkout H071191040"

  14. setelah itu ketik "git commit -m "kumpul tugas 1" "
    "kumpul tugas 1" adalah pesan yang ingin dikirimkan bisa kalimat yang lain

  15. setelah itu ketik "git push origin H071191040"
    pada Windows, apabila muncul gambar seperti ini

    klik "Sign in with your browser"

    setelah berhasil akan tampil seperti berikut

  16. kembali pada halaman repo hasil forking sebelumnya


    klik Pull request

    lalu klik New pull request

    setelah itu ganti master(atau main) menjadi nim


    lalu klik Create pull request

    lalu isi komen pada kolom "Leave a comment"(jika ada)
    setelah itu klik "Create pull request"

  17. Done
